Edenget

a multiplatform batch downloader for mangaeden.com

downloader automatico multipiattaforma per mangaeden.com

Download this project as a compiled windows package Scarica il progetto compilato per windows Download this project as a tar.gz file Scarica il progetto come file tar.gz

EdenGet is a multiplatform batch downloader for mangaeden.com. It runs well on Linux, Windows, Osx and all OS where you can run python2 and gtk+.

EdenGet e' un downloader automatico multipiattaforma per mangaeden.com. Funziona bene su Linux, Windows, Osx e tutti gli OS dove e' eseguibile python2 e le librerie gtk.

Mirrors

This software can be extended with other manga mirrors; if you're interested in implement another mirror, fork this project. Only mangaeden.com is included at the moment.

Questo software puo' essere esteso con altri mirror per manga; se sei interessato nell'implementazione di un nuovo mirror, forka questo progetto. Solo mangaeden.com e' incluso al momento.
